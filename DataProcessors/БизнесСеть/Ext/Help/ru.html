Бизнес-сеть

Представляет собой сервисы для взаимодействия пользователей программ 1С:Предприятие. Одним из таких сервисов является упрощенный механизм обмена электронными документами между организациями без использования электронной подписи.

Основные возможности

Основные возможности
  • отправка электронного документа контрагенту без использования Электронной подписи;
  • получение списка входящих документов;
  • загрузка входящего документа в программу;
  • просмотр профиля контрагента.

При обмене электронными документами используются актуальные форматы обмена — ФНС, CommerceML.

Для подключения программы к сервису 1С:Бизнес-сеть требуется зарегистрировать хотя бы одну организацию в сервисе.
