Карточка

В форме отображается карточка торгового предложения. Выводится информация о товаре, его характеристиках, поставщике, контактных данных поставщика.

Из формы просмотра, по гиперссылкам открывается профиль поставщика, форма поиска товаров по указанной категории товара.

По команде Добавить в корзину товар добавляется в корзину формы поиска, из которой открывалась карточка товара.
