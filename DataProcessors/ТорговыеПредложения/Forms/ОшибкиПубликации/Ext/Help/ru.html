Разделы заполнения

Для удобства заполнения реквизитов предусмотрены следующие разделы:

  • Сведения о товаре
  • Контактная информация
  • Учетные данные
  • Ошибки публикации

Заполнение реквизитов

Для быстрого перехода к заполнению ключевых преквизитов следует нажать на гиперссылку соответствующего объекта. Незаполненные реквизиты будут выделены на открывшейся форме. Количество ошибок и предупреждений указано справа бдля каждой строки.

Повторная проверка

Для повторного выполнения проверки сделует выполнить команду Повторить проверку.
