Состояния публикации

Возможны три состояния публикации товара:

  • Опубликовано – товар успешно опубликован в сервисе и доступен покупателям;
  • Ожидает публикации – товар был изменен по отношению к товару в сервисе;
  • Ошибка – не удалось опубликовать или обновить товар в сервисе.

Для установки отбора требуется нажать на требуемые переключатели состояния публикации.

Только предупреждения

В случае, если флажок установлен, в списке устанавливается отбор только по товаром, содаржащим предупредения.

Отбор по категориям

Категории представляют собой рубрики товаров в сервисе 1С:Торговая площадка Для включения отбора по категориям сервиса требуется выделить отбираемые категории сервиса.

Выявление ошибок и предупреждений

Для перехода к диагностике ошибок и предупреждений следует:

  • выделить анализируемые строки товаров;
  • перейти в контекстное меню;
  • выполнить команду Показать ошибки и предупреждения.

Построчная синхронизация товаров

Построчная синхронизация товаров предназначена для быстрой передачи сведений о конкретных товарах в сервис 1С:Торговая площадка. Для выполнения следует:

  • выделить синхронизируемые строки товаров;
  • перейти в контекстное меню;
  • выполнить команду Синхронизовать выделенные.

Сопоставление товаров

Для перехода к сопоставлению товаров в контекстном меню доступны команды:

  • Выбрать категорию для публикации в 1С:Бизнес-сеть
  • Сопоставить номенклатуру с 1С:Номенклатура
