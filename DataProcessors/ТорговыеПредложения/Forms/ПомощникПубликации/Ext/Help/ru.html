Регистрация в сервисе 1С:Бизнес-сеть

Первый шаг для публикации торговых предложений. По команде Зарегистрироваться в сервисе 1С:Бизнес-сеть открывается форма регистрации организаций. Если ни одна организация не зарегистрирована, появляется предупреждение — Организации не зарегистрированы.

Настройка регионов продаж и доставки

Настройка регионов доставки, адресов самовывоза товаров поставщика для возможности поиска покупателями торговых предложений по региону с отбором по способу доставки.

Подготовка торговых предложений к публикации

По команде Подготовить торговые предложения к публикации открывается форма публикуемых торговых предложений, где можно добавить новые торговые предложения для публикации, указать дополнительную информацию, сопоставить номенклатуру. Если торговые предложения отсуствуют, появляется предпупреждение — Требуется настройка.

Сопоставление номенклатуры

Для публикации товаров по рубрикам сервиса 1С:Бизнес-сети, необходимо сопоставить номенклатуру с карточками 1С:Номенклатура (при настройке подсистемы в программе) с рубрикатором сервиса. Сопоставление с рубрикатором производится по видам номенклатуры (если учет по видам включен в программе), по иерархии номенклатуры, по номенклатуре.

Проверка публикуемых торговых предложение

По команде открывается отчет о всех публикуемых торговых предложениях с информацией по товарам, ценам, дополнительным реквизитам. Если номенклатура для публикации не найдена, появляется предпупреждение — Отсутствуют товары для публикации.

Настройка публикации по расписанию

Для автоматической публикации торговых предложений в сервисе необходимо включить флаг Автоматически публиковать и настроить расписание операции.

Выполнение публикации

Команда Опубликовать торговые предложения выполняет выгрузку торговых предложений в сервис. В случае успешного выполнения будет показываться информация Синхронизация выполнена и время выполнения, в случае возникновения ошибок показывается информация об ошибке. Команда доступна, если во всех предыдущих шагах нет ошибок.
