Обмен с сайтами

Обработка предназначена для автоматического создания сайта в домене UMI.ru.

Создание сайта

Создание сайта
  • Нажать кнопку Создать;
  • Выбрать Тип сайта. Для типа Интернет-магазин в 1С создается узел обмена с настройками;
  • Указать e-mail, на который будет зарегистрирован сайт;
  • Ввести Доменное имя, которое еще незанято. Проверка имени сайта выполняется при заполнении поля ввода.

Для других типов сайтов из 1С передается контактная информация организации.

В дополнительных настройках можно указать организацию, из которой будет получена контактная информация. При создании типа сайта Интернет-магазин возможно выгрузить вид цен товаров, признак отображения остатков на сайте, список групп или категорий товаров.
