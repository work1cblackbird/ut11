Партионный учет

Версия 2.1
  • учет себестоимости товаров в разрезе документов поступления (формирующих учетную партию);
  • расчет себестоимости по методу ФИФО (скользящая оценка);
  • раздельный учет товаров по налогообложению НДС.

Версия 2.2


Предоставляет дополнительные возможности по сравнению с версией 2.1:

  • автоматический переход между методами оценки стоимости По средней стоимости, ФИФО (взвешенная оценка), ФИФО (скользящая оценка);
  • корректный расчет «встречного» выпуска для всех методов оценки стоимости;
  • уменьшение времени расчета за счет уменьшения количества используемых регистров.

Не используется
  • обеспечивает быстрый расчет себестоимости;
  • не поддерживает перечисленные выше возможности режимов Версия 2.1 и Версия 2.2.

Управленческий учет по правилам международного финансового учета

Включение флага "Управленческий учет по правилам международного финансового учета" позволяет дополнительно вести учет себестоимости товаров, доходов и расходов без НДС в валюте управленческого учета.

Использовать управленческий учет по правилам МФУ можно во всех периодах применения партионного учета версии 2.2, либо начиная с указанного периода.
