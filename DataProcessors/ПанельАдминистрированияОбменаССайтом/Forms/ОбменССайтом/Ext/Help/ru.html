Обмен с сайтом

Используется для настройки параметров обмена с сайтами.

  • Команда Создать сайт открывает помощник создания сайта c помощью сервиса 1C-UMI.
  • Команда Настройки обмена с сайтом открывает форму настроек обмена с сайтом.

Описание функциональности в документации на ИТС: https://its.1c.ru/db/utdoc#bookmark:Websites:1cumi
