Панель администрирования 1С:Маркировка

В панели Настройка 1С:Маркировки включается возможность обмена данными в программах 1С:Предприятие с Государственной информационной системой маркировки (ГИСМ).

Для обмена данными с ГИСМ необходимо указать сертификат ключа электронной подписи и шифрования, которым будут подписываться документы, отправляемые в систему ГИСМ. Для электронной подписи и шифрования требуется установить специальную стороннюю программу (криптопровайдер).

Электронная подпись - реквизит электронного документа, с помощью которого можно удостовериться в неизменности электронного документа с момента его подписи. Для установки подписи необходимо иметь сертификат ключа электронной подписи. Сертификат можно получить в удостоверяющем центре. Например, в удостоверяющем центре фирмы 1С http://ca.1c.ru/. Только удостоверяющий центр может выдать юридически значимый (квалифицированный сертификат).

Предусмотрена возможность автоматического обмена данными с ГИСМ. Можно настроить расписание для автоматического обмена данными с системой ГИСМ. Если обмен будет производиться автоматически, то необходимо указать пароль. При обмене в ручном режиме пароль следует указывать при каждом обмене.
