Форма команд администратора

Форма предназначеня для ручной выгрузки данных и получения дополнительной информации об обученных моделях прогнозирования.
На закладке Выгрузка данных имеется возможность отметить коллекции для выгрузки в сервис прогнозирования, и затем произвести выгрузку одной из следующих команд:

  • Выгрузка (мягкая) - производится полная выгрузка с сохранением исходных идентификаторов коллекций.
  • Выгрузка (жесткая) - производится полная выгрузка с удалением коллекций по исходным идентификаторам. Выгруженным коллекциям назначаются новые идентификаторы.

Информация о выгрузке и принятии сервисом данных даны для ознакомления, и не гарантируют актуальности.
Для получения актуальных данных, необходимо перейти на закладку Информация о данных на сервере.
Информация для этой закладки получается не только по данным информационной базы, но и запрашивается на сервере. Таким образом, можно увидеть, какое количество информации по интересующим коллекциям имеется на сервере сервиса прогнозирования продаж.
