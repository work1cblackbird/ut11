Подбор товарных категорий по отбору

Рабочее место предназначено для подбора товарных категорий в документ в соответствии с установленными параметрами отбора.

Порядок работы с обработкой

Порядок работы с обработкой
  • Установите нужные параметры отбора. В список параметров отбора можно добавить дополнительные параметры из списка доступных полей.
  • Нажмите на кнопку Заполнить. В список Подобранные товарные категории будут добавлены те товарные категории, которые удовлетворяют установленным параметрам отбора.
  • Уточните список подобранных товарных категорий (добавьте/удалите новые товарные категории в сформированный список)
  • Нажмите на кнопку Перенести в документ для переноса отобранных товарных категорий в табличную часть документа.
