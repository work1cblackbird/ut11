Оформляемые документы

Отчет предназначен для отработки результатов оформления актов о расхождениях.
Отчет формируется в зависимости от выбранного в документе способа отражения расхождений, и варианта действий по каждой строке с расхождениями.
Для каждого необходимого шага по отработке расхождений можно вывести подробно описание какой документ должен быть создан или изменен и почему. Для этого необходимо установить флаг "Выводить легенду" и переформировать отчет.
Отчет выводится с группировкой по уровням. Рекомендуется выполнять нижестоящие действия после вышестоящих.

В отчете возможно будут предложены следующие действия:

  • Изменение документа - выводится гиперссылка, при нажатии на которую изменяется первичный документ, по которому выявлены расхождения.
  • Оформление нового документа - выводится гиперссылка, при нажатии на которую создается новый документ, заполненный на основании акта. Если требуемый документ уже создан, то вместо команды "Оформить" выводится гиперссылка на ранее созданный документ.
