Обработка "Переход на синхронизацию данных через универсальный формат"

Обработка осуществляет переход:

  • с существующей настройки синхронизации данных между конфигурациями "Управление торговлей", ред. 11.1 и "Бухгалтерия предприятия" либо "Бухгалтерия предприятия КОРП", ред. 3.0
  • на синхронизацию данных через универсальный формат.

 

В результате перехода создается новая настройка "Синхронизации данных через универсальный формат". При этом из "старой" настройки в "новую" копируются:

  • Все ключевые настройки узла (префикс корреспондента, настройки отправки справочников и документов, отбор по организациям и т.д.)
  • Регистрация изменений
  • Настройки транспорта сообщений
  • Соответствие объектов информационных баз
  • Сценарии обмена данными
  • Дата запрета загрузки
  • Предупреждения при обмене данными

 

Запуск обработки выполняется:

  • либо автоматически при обновлении на версию "Управление торговлей" 11.1.11. Автоматический запуск происходит, если в существующей настройке синхронизации используются типовые правила обмена
  • либо вручную из формы узла настройки синхронизации данных (меню "Параметры синхронизации данных" - пункт "Переход на синхронизацию данных через универсальный формат"). Ручной запуск выполняется по решению администратора системы.

 

После завершения процедуры перехода на синхронизацию данных через универсальный формат необходимо выполнить синхронизацию данных с базой-корреспондентом. При этом в базе-корреспонденте автоматически выполнится переход настройки синхронизации на синхронизацию данных через универсальный формат.

Внимание. Операция перехода является необратимой. Перед выполнением операции необходимо создать копию всех информационных баз, которые участвуют в синхронизации.
