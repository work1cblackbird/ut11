Цены (прайс-лист)

Обработка предназначена для просмотра и изменения цен номенклатуры. Регистрация цен на номенклатурные позиции по конкретному виду цен - это формирование определенной колонки прайс-листа.

Прайс-лист можно сформировать на дату или на определенный период. Формирование прайс-листа на период позволяет задать временные (акционные) цены. Обычно, цены, заданные на период, привязаны к маркетинговой акции. После окончания периода цены становятся прежними.

Формирование прайс-листа осуществляется по кнопке Сформировать.

Настройка параметров прайс-листа


Для просмотра и регистрации цен нужно нажать на кнопку Сформировать.

В форме настройки необходимо установить параметры отбора номенклатуры (например, Цена установлена, Маркетинговое мероприятие). Если параметры не установлены, то будет заполнен весь список номенклатуры.

Используя вариант настройки Расширенный вид можно устанавливать более сложные варианты отбора с группировкой условий (и, или, максимум, минимум), возможностью отбора по любым реквизитам номенклатуры и т.д.

Установленные настройки отбора сохраняются на следующий сеанс работы.

На закладке Колонки прайс-листа настраивается список тех видов цен, которые надо назначить. Можно выбрать как вид цены с возможностью изменения, так и без возможности изменения (только просмотр).
По каждому виду цены показывается информация о дате последнего изменения цены.
Можно показать только выбранные для изменения и/или просмотра виды цен.

Для каждой позиции номенклатуры в сформированном списке можно задать цены вручную. В ячейках с ценами, измененными вручную отображается специальная пиктограмма (карандаш). Если цена увеличилась - цена выделяется синим цветом, если уменьшилась - зеленым цветом.
При изменении цены, зависимые от нее цены будут автоматически пересчитаны. После пересчета зависимые цены автоматически округляются в соответствии с теми правилами округления, которые заданы для вида цены.

На закладке Сортировка и группировка можно задать поля номенклатуры (код, артикул и другие), по которым будет отсортирован список выводимой в прайс-лист номенклатуры.
Также предусмотрена возможность сортировки видов цен.

Цены в прайс-листе можно изменять. Для этого нужно либо воспользоваться непосредственным изменением цен в ячейках, либо изменить цены при помощи специальных команд для группового изменения цен: Рассчитать вычисляемые цены (по данным ИБ и формулам), Загрузить из документов установки цен, Загрузить действующие цены, Изменить цены, Загрузить из Яндекс.Маркет и Загрузить из Ozon.

Если в настройках отбора в карточке Виды цен установлено Запретить установку цены за пределами отбора, то программой устанавливается запрет на редактирование цены.

Настройка параметров выводимой информации

Предусмотрена возможность настройки дополнительных параметров выводимой информации, доступная по гиперссылке Настроить:

  • Показывать Действующие цены на - Показывать сумму наценки и/или Показывать процент наценки - будут выведены колонки с действующей ценой до изменения и процентом/суммой наценки.
  • Формулы - будут выведены колонки с формулами расчета для каждой текущей цены.
  • Панель изменения цен - на форму прайс-листа будет выведена панель с опциями для возможности быстрого редактирования цен по каждой выделенной ячейки в форме прайс-листа и с данными по истории изменения цены. В истории изменения цены можно видеть, на какие конкретно даты было зафиксировано изменение цены. При необходимости из формы истории изменения цены можно сформировать Отчет по динамике цен.
  • Рассчитывать автоматически - позволяет рассчитывать цены в строке автоматически при изменении номенклатуры и влияющих цен.
  • Отображать влияющие цены - показываются влияющие цены (в режиме просмотра, для информации).
  • Зафиксировать колонку номенклатуры - позволяет отображать постоянно на форме информацию о номенклатуре.
  • Показывать информацию по номенклатуре в одной колонке - позволяет объединить все разрезы одной номенклатурной позиции из разных колонок в одну.

Регистрация цен


С помощью гиперссылки История изменения цен номенклатуры можно посмотреть список ранее зарегистрированных документов Установка цен номенклатуры. Используя команду Прайс-лист можно сформировать прайс-лист по отобранным в списке позициям.

Предусмотрена также возможность регистрации цен для конкретного товара из справочника Номенклатура.

Для сохранения измененных в прайс-листе цен нужно нажать на кнопку Применить измененные цены, при необходимости указать комментарий и выбрать режим записи создаваемого документа установки цен (Записать или Записать и Провести). При выполнении этой команды будет автоматически создан документ Установка цен номенклатуры со списком зарегистрированных цен.

Анализ зарегистрированных цен


Анализ всех зарегистрированных видов цен, используемых при продажах товаров, может быть осуществлен по гиперссылке Прайс-лист.
