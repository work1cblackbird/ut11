Мобильное рабочее место кладовщика

Мобильное рабочее место предназначено для упрощения процессов оформления складских операций при помощи терминала сбора данных (ТСД) или другого мобильного устройства (смартфона).

Использование мобильного рабочего места позволяет оформлять операции приема, размещения, перемещения, отбора, проверки отбора и пересчета товаров.

Мобильное рабочее место кладовщика может использоваться только на складе (помещении) с ордерным или адресным хранением товаров.

Преимущества использования мобильного рабочего места:

■ уменьшение количества ошибок при выполнении складских операций
■ увеличение скорости выполнения складских операций
■ работник склада может оформлять все операции с мобильного устройства, без использования компьютера

Особенности использования мобильного рабочего места:

■ предназначено для рядового сотрудника, задания которому выдает диспетчер, работающий за стационарным рабочим местом, а также для самостоятельной работы с мобильного устройства
■ система контролирует действия работника, а также подсказывает, какие действия ему необходимо выполнить на складе

Для рабочего места работника склада используется технология мобильного клиента.

Мобильный клиент работает на ОС Android и IOS.
Мобильный клиент поддерживает работу с подключаемым оборудованием (сканер ШК).

Функциональные возможности

Приемка товара на ордерный склад

- Доступ к просмотру распоряжений на приемку
Список с распоряжениями, которые служат основанием для начала приемки товара на склад

- Создание документа "приходный ордер на товары"
Проведенный документ фиксирует факт приема товара на ордерный склад

Размещение товаров по ячейкам

- Набор товаров для размещения в зоне приемки
Форма позволяет кладовщику набрать товары из зоны приемки для последующего размещения по ячейкам

- Создание документа "задание на размещение товаров"
Проведенный документ забирает товары из зоны приемки и размещает их по ячейкам адресного склада

Отгрузка товаров с ордерного склада

- Доступ к просмотру распоряжений на отгрузку
Список с распоряжениями, которые служат основанием для начала отгрузки товара со склада

- Создание документа "расходный ордер на товары"
Созданный документ в статусе "К отбору" является основанием для начала отбора товара из ячеек адресного склада

- Заполнение ордера на основании отобранных товаров из ячеек
После завершения отборов товара из ячеек регистрирует факт отгрузки товара с ордерного склада

Отбор товаров из ячеек

- Создание документа "задание на отбор товаров"
Проведенный документ забирает товары из ячеек

Перемещение товаров

- Создание документа "задание на перемещение товаров"
Проведенный документ перемещает товары между ячейками одного склада.
Поддерживается функция переупаковки. Можно отобрать товар в упаковке "коробка (10шт)", а разместить его в упаковке "упак (1 шт.)"

Пересчет товаров

- Работа с созданным документом "пересчет товаров"
Форма позволяет заполнить созданный документ пересчета фактическим количеством товара на складе

Блокировка ячеек

- Создание документа "установка блокировки ячеек"
Форма позволяет кладовщику блокировать и разблокировать ячейки

Поиск товаров

Форма для поиска товара. Показывает остатки на складах и в ячейках.

Остатки товаров в ячейке

Форма показывает остатки товаров в выбранной ячейке.
