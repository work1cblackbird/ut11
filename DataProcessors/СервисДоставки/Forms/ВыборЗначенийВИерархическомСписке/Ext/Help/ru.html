Выбор значений в иерархическом списке

Необходим для фильтрации списка заказов на доставку по списку выбранных состояний.
Содержит список всех состояний заказов сервиса, представленный в виде иерархического дерева. Состояния сгруппированы по этапам работы с заказом. Можно выбрать несколько состояний и групп состояний.
Для применения фильтра необходимо нажать на команду ОК.
