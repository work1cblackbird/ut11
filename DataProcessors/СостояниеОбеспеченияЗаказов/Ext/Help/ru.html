Состояние обеспечения заказов

Рабочее место предназначено для контроля обеспечения номенклатурных позиций заказов на отгрузку.

В колонке Состояние предусмотрены следующие значения:

В резерве - отображается для номенклатурных позиций заказа, для которых в заказе установлено действие Резервировать на складе. А также для позиций с действием Отгрузить, если заказ находится в предварительном статусе, когда оформление документов отгрузки по заказу еще недоступно.
Обеспечен на складе - отображается для номенклатурных позиций заказа, для которых в заказе установлено действие К обеспечению и товар для обеспечения данной позиции поступил на склад (зарезервирован по заказу)
Обеспечен к дате - отображается для номенклатурных позиций заказа, для которых в заказе установлено действие К обеспечению и товар для обеспечения данной позиции имеется в ожидаемом поступлении товаров на дату по заказам на поступление
Ожидается по неподтвержденному заказу - отображается для номенклатурных позиций заказа, для которых в заказе установлено действие К обеспечению и товар для обеспечения данной позиции имеется в запланированном поступлении товаров по заказам на поступление, дата поступления которых не определена, не согласована с поставщиком.
Обеспечить - отображается для номенклатурных позиций заказа, для которых в заказе установлено действие К обеспечению и товара для обеспечения данной позиции нет ни на складе, ни в запланированных поступлениях, либо товар есть, но он зарезервирован по другим заказам на отгрузку.
Отложить обеспечение - отображается для номенклатурных позиций заказа, для которых в заказе установлено действие К обеспечению и указана дата отгрузки позже чем граница периода обеспечения по способу обеспечения потребностей, установленному для товара, по схеме обеспечения, указанной в номенклатуре. Такие позиции не обеспечиваются до момента, когда дата отгрузки не сравняется с границей обеспечиваемого периода.
Не обеспечивать - отображается для номенклатурных позиций заказа, для которых в заказе установлено действие Не обеспечивать.

Для позиций обеспеченных ожидаемыми поступлениями в колонках Дата поступления и Заказ на поступление отображается информация о заказе на поступление. Для необеспеченных позиций в этих колонках отображается способ обеспечения потребностей, используемый для товара и дата поставки по способу обеспечения потребностей.

В рабочем месте предусмотрен отбор строк по состоянию. Отбор вида Ожидается не в срок отбирает позиции заказа, в которых указана дата отгрузки и согласно данным обеспечения позиция будет обеспечена позже этой даты отгрузки.

Если рабочее место используется из формы документа, то доступны команды изменения действий в заказе: Отгрузить и Не обеспечивать. Результаты выполнения действий переносятся в форму документа. В подвале формы отображается контекстная информация о доступности выбранной позиции заказа. Доступность товара на складе и по графику (в ожидаемых поступлениях). Гиперссылка Подробнее о доступности позиции позволяет для выбранной позциии заказа получить список заказов которыми данная номенклатурная позиция зарезервирована.

В рабочем месте отображаются товары и аналогично многооборотная тара и работы.
