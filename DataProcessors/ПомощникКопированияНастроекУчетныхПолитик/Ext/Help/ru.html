Помощник копирования настроек учетных политик

Помощник предназначен для копирования действующих настроек учетных политик организаций.

Копирование настроек доступно в двух режимах: Из другой организации и В другие организации

Для каждого режима используется следующая последовательность действий:

Копирование из другой организации.

  • Выбрать организацию - источник копирования настроек учетной политики.
  • Выбрать настройки, которые требуется скопировать

В другие организации.

  • Выбрать настройки, которые требуется скопировать.
  • Выбрать организации - приемники для копирования настроек учетной политики.
