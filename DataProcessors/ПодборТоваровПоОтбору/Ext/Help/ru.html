Подбор товаров по отбору

Форма предназначена для подбора позиций в документ в соответствии с установленными параметрами отбора.

Порядок работы:
  • Установите нужные параметры отбора. При необходимости уточнив дополнительные параметры из списка доступных полей.
  • Нажмите на кнопку Заполнить и в список Подобранные товары будут автоматически добавлены те товары, которые удовлетворяют установленным параметрам отбора.
  • Уточните список подобранных товаров (удалив/добавив новые товары).
  • Нажмите на кнопку Перенести в документ для переноса отобранных товаров в табличную часть документа.

Пример. Необходимо отобрать товары, которые имеются в наличии на складе Центральный и относятся к ценовой группе Мебель.

Для отбора товаров необходимо установить следующие значения параметров:

  • Установить флаг и указать значение параметра: Номенклатура.Ценовая группа равно Мебель.
  • Установить флаг в поле Только в наличии.
  • Установить флаг и указать значение параметра: Склад равно Центральный.

Если учет номенклатуры ведется в разрезе характеристик, то такая номенклатура попадает в отбор в разрезе конкретных характеристик. Отобрать такую номенклатуру не указав конкретную характеристику невозможно.
