Подбор номеров ГТД в документы

Рабочее место предусмотрено для регистрации номеров ГТД вручную при оприходовании товаров. Форма рабочего места схожа с формой элемента справочника номера грузовых таможенных деклараций, однако имеет некоторые интерфейсные отличия для типа Номер РНПТ комплекта:

  • Над табличной частью выводятся сведения о товаре из накладной, для которого создается номер РНПТ комплекта.
  • Отображается колонка Количество по РНПТ (в документе), в которой указывается приходуемое количество товара по накладной (источником информации может выступать, например, счет-фактура, полученная от контрагента).
  • Значение колонки Количество РНПТ (в комплекте) рассчитывается автоматически, как нормативное значение на единицу комплекта и может быть изменено вручную.

При нажатии на команду Ок выполняется проверка корректности заполнения указанных сведений, по аналогии с формой элемента справочника Номера ГТД.

Если по введенным сведениям в информационной базе будет найден аналогичный элемент, тогда он будет использован в качестве указываемого значения в накладной. Иначе, будет создан новый.

При вычислении нормативного количества комплектующей, приходящегося на единицу комплекта, может возникнуть ошибка округления, проявляющаяся при списании прослеживаемых комплектов. Для исключения возникновения таких ситуаций создаются два элемента номера РНПТ комплекта – основной и остаточный.

Строка товара табличной части документа, для которой осуществляется создание номера РНПТ комплекта, дробится на две строки и для каждой указывается соответственно созданный элемент справочника.

В документе оприходования колонка Количество по РНПТ становится недоступной и на нее накладывается условное оформление с текстом . Также автоматически рассчитывается значение реквизита Количество по РНПТ, которое становится равным значением реквизиту Количество.
