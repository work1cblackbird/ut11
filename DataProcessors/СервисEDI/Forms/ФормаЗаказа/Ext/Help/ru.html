Назначение формы

Форма предназначена для просмотра присланных на согласование документов через сервис 1С:EDI без загрузки данных в учетные документы. Также форма позволяет изменять статус документа, просматривать и сравнивать различные версии документа или загрузить присланный документ в информационную базу.

Загрузка документа

Кнопка "Загрузить" создает новый учетный документ и заполняет его в соответствии с данными электронного документа. После создания документа будет открыт созданный документ.

Панель состояния EDI

Панель состояния предназначена для отображентия сведений о текущем статусе документа в сервисе 1С:EDI. Также в панели отображаются доступные действия документа в сервисе. Доступные действия зависят от текущего статуса документа.

Протокол EDI

В протоколе отображаются:

  • Данные о событиях изменения статуса документа;
  • Пользовательские сообщения, отправленные участниками согласования.

В случае, если по документу есть непрочтенные сообщения, то количество непрочтенных сообщений будет отражаться на кнопке открытия протокола. Для отправки произвольных сообщений, в протоколе предусмотрено поле ввобда и кнопка отправки сообщения в нажней части протокола.

Шапка документа

В шапке документа расположена информация о контрагенте. Контактная информация контрагента недоступна для редактирования, так как устанавливается в процессе согласования документа контрагентом.

Страница "Товары"

Страница содержит в себе сведения о поставляемых товарах и устугах.
Табличная часть содержит следующие колонки:

  • N - номер строки документа;
  • Номенклатура - наименование товара или услуги;
  • Ед. изм - наименование упаковки;
  • Количество - количество заказываемых упаковок;
  • Цена без НДС - цена за единицу товара до применения налога;
  • Скидка - общая сумма скидки по строке;
  • Сумма без НДС - общая сумма по строке до применения налога;
  • Станка НДС - используемая ставка НДС;
  • Сумма НДС - сумма налога по строке;
  • Сумма с НДС - итоговая сумма строки с учетом налога;
  • Дата поставки - планируемая дата поставки товара.

Для контроля суммовых показателей в подвале указаны скидка и общая сумма с учетом налога по документу.

Страница "Оплата и доставка"

Группа "Доставка"
  • Способ доставки - способ доставки товара (Доставка / Самовывоз);
  • Адрес доставки - сведения об адресе доставки, доступен в случае, если выбран способ доставки "Доставка".

Группа "Оплата"
  • Форма оплаты - способ оплаты документа (Наличная / Безналичная / Платежная карта);
  • Банковский счет поставщика - расчет счет поставщика для взаиморасчетов по документу, доступен в случае, если выбрана форма оплаты Безналичная.

Группа "График оплаты"

Группа содержит сведения о этапах оплаты документа. Возможны следующие способы ввода сведений о графике оплаты:

  • Простой - предоплата и кредит;
  • Списком - доступен ввод произвольного количество этапов графиков оплаты.

Страница "Дополнительно"

Группа "Информация о заказе"

Группа содержит сведения нумерации документа с нашей стороны и со стороны контрагента, сведения о валюте взаиморасчетов и способе указания НДС.

Группа "Менеджер"

Группа содержит сведения контактной информации об ответственном сотруднике за документ со с нашей стороны. Редактирование данного раздела для контрагента недоступно.
