Настройка отбора по статусам

Данная форма предоставляет возможность выполнить детальную настройку отбора по состоянию EDI в формах списков прикладных документов.
Если отбор по статусам будет использован в списке, то необходимо установить флаг Использовать отбор по статуса EDI
После установки флага появится возможность выбрать статусы, по которым требуется установить отбор.
В дополнение к отбору по статусам есть возможность установить отбор по документам, с проблемами синхронизации между информационной базой и сервисом.
В такой отбор попадут документы, для которых либо в сервисе есть более актуальная версия, либо не удалось передать информацию об изменениях в сервис.
