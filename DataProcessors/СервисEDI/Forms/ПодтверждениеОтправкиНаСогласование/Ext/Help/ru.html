Назначение формы

Форма предназначена для подтверждения изменений заказа, отправляемых в сервис 1С:EDI для согласования контрагентом. Форма отображается пользователю, когда отправляются изменения по согласуемым реквизитам документа. В случае, изменения реквизитов, согласование которых не предусмотрено в сервисе, данная форма отображаться не будет.

Просмотр изменений

Для просмотра печатной формы сравнения версий документа требуется нажать на гиперссылку "Показать изменения". Сервис 1С:EDI сформирует печатную форму сравнения текущего состояния документа с версией, присанной на согласование контрагентом.

Настройки показа формы

Пользователь может управлять показом подтверждения. В случае, если пользователь установит флаг "Больше не показывать и отправлять на согласование сразу" даннаф форма не будет отображаться для данного пользователя.

Принятие решения

В случае, если пользователь согласен передать контрагенту на согласование изменения, указанные в отчете о сравнении, то пользователь нажимает кнопку "Отправить на согласование". Кнопка "Отмена" вернут пользователя к редактированию документа, при этом документ не будет отправлен контрагенту.
