Назначение формы

Форма предназначена для быстрого отслеживания событий по документам с возможностью быстрых отборов.

Панель отборов

На панели отборов расположены следующие элементы:

  • Кнопка "Обновить" предназначена для повторного получения списка событий из сервиса.
  • Отбор по периоду событий - позволяет отобрать события за один из стандартных периодов.
  • Кнопка "Выбрать период" - позволяет указать произвольный период отображения событий.
  • Поле "Организация" - позволяет отфильтровать события для одной из органищаций информационной базы.
  • Поле "Менеджер" - позволяет отфильтровать события по ответственному собтруднику документа.
  • Гиперссылка направления документов - возволяет отобрать документы продажи или документы закупки.

Список событий

Список содержит следующие колонки:

  • Дата - дата события
  • Организация - организация документа
  • Кто / кому - направление события
  • Контрагент - контрагент документа
  • Событие - содержимое события
  • Документ - представление ключевых свойств документа
  • Статус - состояние документа после события

Список сортируется от по убыванию даты (от новых событий к старым) и обновляется автоматически при изменении отборов.

В случае, если были загружены не все события, под списком отображается команда "Загрузить еще".

Для быстрого отбора событий по определенному документу предусмотрена команда контекстного меню "Показать события по документу". При установленном отборе в контекстом меню становится доступной команда "Отменить отбор по документу" для возвращения к общему списку событий.
