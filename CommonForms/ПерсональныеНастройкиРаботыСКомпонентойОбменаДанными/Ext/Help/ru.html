Настройки компоненты обмена данными с КПК

В форме можно определить настройки компоненты для текущего пользователя.

Подключение компоненты обмена данными

Подключение компоненты обмена данными

Для использования компонента обмена данными она должна быть подключена. Подключение можно выполнять один раз во время сенса работы. Также можно установить флаг Подключать компоненту обмена данными при старте системы, В этом случае компонента будет автоматически подключаться каждый раз при начале работы пользователя с системой.
