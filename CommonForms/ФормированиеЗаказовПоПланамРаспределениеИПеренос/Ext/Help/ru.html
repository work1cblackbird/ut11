Распределение и перенос

Форма предназначена для распределения потребностей в планируемой номенклатуре на другие даты или несколько периодов в рамках периода планирования. Это позволяет сформировать по одной строке плана несколько заказов с разными периодами.

Перенести дату потребности планируемой номенклатуры

Позволяет перенести дату потребности планируемой номенклатуры по правилу для каждой строки плана.

Варианты переноса:

  • На число месяца года/полугодия/квартала/месяца/декады/недели - позволяет перенести на указанный день недели, число месяца, номер дня от начала года/полугодия/квартала. Если указано 31 число месяца - будет установлен последний день месяца.
  • На первый день - позволяет перенести на первый выбранный день в периоде строки плана. Дата в каждом периоде зависит от выбранного дня недели. Например, если период планирования Месяц, и выбран день Среда, то будет установлена дата первой среды в месяце.
  • На последний день, аналогично переносу на первый день.

Распределить количество планируемой номенклатуры

Позволяет распределить количество в рамках текущего периода планирования.

Способ распределения планируемого количества номенклатуры в периоде:

  • Пропорционально - количество будет разделено равномерно в периоде.
  • По долям - на следующей странице необходимо указать доли для каждого периода распределения.

Варианты распределения периода:

  • По полугодиям - количество будет распределено на 2 части. Даты поступления будут установлены на начало каждого полугодия.
  • По кварталам - количество будет распределено на 2 части, если периодичность сценария "Полугодие" или на 4 части, если периодичность сценария год.
  • По месяцам - распределение на 12, 6 или 3 части в зависимости от периодичности сценария.
  • По декадам - даты поступления будут установлены на 1, 11 и 21 числа в одном периоде планирования.
  • По неделям - количество недель зависит от периодичности сценария и выбранного дня недели для поступления.
  • По количеству периодов - указывается произвольное количество частей распределение. Но не может быть больше количества дней в периода планирования.
