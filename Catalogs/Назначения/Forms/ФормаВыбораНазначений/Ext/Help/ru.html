Форма выбора назначений

На форме выбора назначений может быть отображен ряд флажков. С помощью них список назначений может быть отфильтрован по данным учета.
Например, при установке флажка Потребность на складе "..." появится одноименная колонка, а список уменьшится до тех строк, для которых есть потребность.
При снятых флажках, либо когда флажки не отображаются, доступны все назначения.
В каждом документе могут быть заданы дополнительные условия отбора, позволяющие выбрать только допустимые назначения. Например, если в документе задано Направление деятельности, тогда будут показаны только те назначения, у которых задано это направление деятельности.

Отображаемые флажки и колонки могут содержать следующие показатели:

  • Потребность на складе "..." - количество заданной номенклатуры, которое необходимо обеспечить обособленно на определенном указанном складе.
  • Потребность на всех складах - количество заданной номенклатуры, которое необходимо обеспечить обособленно на всех складах.
  • Потребность в подразделении "..." - количество заданной номенклатуры, которое необходимо обеспечить обособленно в определенном подразделении.
  • В наличии на складе "..." - количество заданной номенклатуры, которая уже поступила на определенный склад.
