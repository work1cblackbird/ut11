Добавление новых условий обслуживания


В форме можно указать список клиентов, для которых требуется задать условия обслуживания. Для каждого из них будут созданы указанные условия обслуживания.

Выбор соглашения


Если в списке клиентов, для которых создаются условия обслуживания, указано более одного клиента, то могут быть выбраны только типовые соглашения. Если же клиент один, то может быть выбрано как типовое, так и индивидуальное соглашение с указанным клиентом.
