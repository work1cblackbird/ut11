Правила обмена с подключаемым оборудованием offline

Правила обмена используются при работе с кассами ККМ и электронными весами, которые работают в режиме OffLine. С помощью правил обмена настраивается список товаров для выгрузки данных в кассу ККМ и электронные весы, назначаются соответствующие коды для работы с подключаемыми устройствами в удаленном режиме.

По правилам обмена осуществляется загрузка перечня товаров на кассы (весы), работающие в режиме Offline. Для каждой кассы (весов) должно существовать минимум одно правило. Одно и тоже правило может быть назначено нескольким кассам (весам).

Настройка правила обмена


В карточке необходимо в поле Тип подключаемого оборудования выбрать значение ККМ Offline для кассы или Весы с печатью этикеток, если осуществляется настройка для электронных весов. Количество товаров выгружаемых в кассу (весы) может быть ограничено, при помощи поля Максимальное количество товаров. Также в правиле можно установить отбор номенклатуры, например, загружать только новые позиции - флаг Выгружать изменения.
В случае выбора в качестве типа подключаемого оборудования Весы с печатью этикеток, дополнительно потребуется уточнить в правиле обмена единицу измерения веса. В результате в загрузке будут принимать участие товары, только с установленной единицей измерения.

Перечень товаров участвующих в обмене


По гиперссылке Показать список товаров для редактирования можно открыть список товаров участвующих в обмене с ККМ или весами (в соответствии с установленным фильтром). В списке можно отредактировать автоматически назначенный код товару (под которым товар будет загружен в кассу или весы). По команде Администрирование – Удалить свободные коды в конце списка из конца списка будут удалены строки с пустой номенклатурой. При заполнении списка производится проверка наличия у товара штрих-кода и значения цены. Значение цены проверяется по тому типу цен, который указан в карточке выбранного склада. Если для отобранных товаров не установлены цены или штрих-код, то информация об этом отображается в соответствующих колонках списка. Если в правиле задано ограничение списка по максимальному коду, то товары с кодом превышающим максимальный участвовать в обмене не будут и будут выделены в списке красным цветом. По команде Обновить список товаров участвующих в обмене с ККМ будет записан.
