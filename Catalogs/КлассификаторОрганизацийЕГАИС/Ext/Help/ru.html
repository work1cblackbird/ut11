Классификатор организаций ЕГАИС

Классификатор предназначен для хранения сведений о поставщиках, производителях и импортерах алкогольной продукции в системе ЕГАИС. Классификатор ведется на федеральном уровне. Новые элементы добавляются в классификатор при загрузке документов из ЕГАИС, при загрузке классификатора алкогольной продукции, при запросах по коду ФСРАР или по ИНН.

В поле Формат обмена нужно указать версию формата схем обмена с ЕГАИС, который поддерживает выбранная организация. Для собственных организаций есть возможность выгрузить информацию об изменении формата в ЕГАИС, воспользовавшись командой с пиктограммой

Элементы классификатора должны быть сопоставлены с поставщиками (партнерами, импортерами) и их контрагентами или с организациями предприятия. Для этого предназначен переключатель Соответствует.

Если организация находится в сельской местности, нужно установить флаг Находится в сельской местности в карточке классификатора. Для такой организации применяется другая дата начала отражения данных в системе ЕГАИС, задаваемая в настройках программы в соответствующей константе.
