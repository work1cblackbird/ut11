Установка минимально допустимых цен продажи


В поле Вид минимально допустимых цен продажи выберите из списка зарегистрированных видов цен тот вид цен, который будет использоваться, как минимальный вид цен продажи. Цены, зарегистрированные по данному виду цен, будут являться теми минимальными ценами, ниже которых не допускается продавать товар.

Установка максимально допустимых цен продажи


В поле Вид максимально допустимых цен продажи выберите из списка зарегистрированных видов цен тот вид цен, который будет использоваться, как максимальный вид цен продажи. Цены, зарегистрированные по данному виду цен, будут являться теми максимальными ценами, выше которых не допускается продавать товар.

Контроль допустимых цен продажи не производится, если для пользователя установлены Полные права или если у пользователя установлена роль, которая допускает отклонение от условий продаж.

Установка максимально допустимых цен закупки


В поле Вид максимально допустимых цен закупки выберите тот вид цен, который будет использоваться в качестве максимально допустимых цен закупки. Цены, которые зарегистрированы для товаров по данному виду цен, будут контролироваться при оформлении документов, сопровождающих поступление товаров: соглашений с поставщиками, заказов поставщиков, документов поступления.

Контроль допустимых цен закупки не производится, если для пользователя установлены Полные права или если у пользователя установлена роль, которая допускает отклонение от условий закупок.
