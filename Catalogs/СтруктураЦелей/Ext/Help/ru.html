Структура целей

Справочник предназначен для регистрации целей и целевых показателей в системе.
Целевые показатели образуют иерархическую структуру, которая отражает логическую или математическую зависимость одних целевых показателей от других. На верхнем уровне структуры находятся укрупненные цели, которые предприятие планирует достичь в будущем. Каждая из укрупненных целей состоит из множества подцелей, успешное выполнение которых и обеспечивает достижение основной цели.

Система поставляется с заполненной структурой целевых показателей и списком вариантов для их анализа.
Доступна возможность по уточнению параметров анализа, добавлению новых целевых показателей, а также созданию собственной структуры целевых показателей.
Настройку структуры целевых показателей и вариантов анализа производит администратор системы в соответствии с подготовленной руководством предприятия стратегией.

Настроенная структура целевых показателей используется для предоставления информации о деятельности предприятия в мониторе целевых показателей.

Расчет числовых показателей

Все числовые показатели (ресурсы/итоги) в СКД для МЦП рассчитываются двумя различными способами:

  • По периодам
  • По аналитикам (объектам анализа)


В СКД это можно наблюдать на закладке "Ресурсы" - в колонке "Рассчитывать по...", там указано для каких полей используется "Выражение".
