Настройка доступности

В форме отображается список пользователей, сгруппированный по подразделениям.

Для того чтобы вариант анализа стал доступен пользователю следует отметить соответствующего пользователя. Для предоставления доступа всем сотрудникам подразделения следует отметить соответствующее подразделение.

Для быстрого снятия или установки отметки можно использовать соответствующие кнопки командной панели - Отметить все и Снять отметки.

Для быстрого отбора пользователей, которым предоставлен доступ к варианту анализа следует использовать кнопку Только выбранные пользователи. В списке будут отображены отмеченные пользователи.
