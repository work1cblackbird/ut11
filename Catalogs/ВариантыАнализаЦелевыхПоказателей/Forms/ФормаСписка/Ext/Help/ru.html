Варианты анализа целевых показателей

В справочнике хранится информация о вариантах анализа, которые используются для анализа целевых показателей. Для каждой цели используется свой список вариантов анализа целевых показателей.

В форме списка пользователь, имеющий соответствующие права, может добавлять не только новые варианты анализа целевых показателей, но и вводить информацию о новых целях в структуру целей. Для дополнительной детализации данных могут быть также использованы различные варианты отчетов конфигурации.

Дополнительные материалы на ИТС

Список вариантов анализа целевых показателей разделен на две части:

  • В левой части списка показывается структура целей.
  • В правой части для каждой цели (целевого показателя) показывается список вариантов, предназначенных для анализа выбранного целевого показателя.

В варианте анализа заполняется информация о типе анализа, периоде анализа, периодичности контроля. Результаты анализа могут быть представлены в кратком виде, в виде диаграммы или таблицы. Для каждого варианта анализа могут быть заданы целевые значения, с помощью которых будет осуществляться контроль целевого показателя. Предусмотрена возможность прогнозирования данных, а также сравнение с аналогичными данными прошлых периодов. Для показателя можно настроить список отчетов для расшифровки данных, а также определить список тех пользователей, которым будет доступен просмотр данного показателя.
