Перечень Излишки, недостачи и порчи товаров

В списке Излишки, недостачи и порчи товаров показывается список оформленных ордеров на отражение излишков, недостач и порчи товаров.

Информация в списке показывается по отношению к конкретному складу (помещению). Перед просмотром информации в списке необходимо в поле Склад выбрать склад, по которому необходимо просмотреть или оформить ордера на отражение излишков и недостач товаров. Если склад разделен на помещения, то необходимо указать конкретное помещение. В качестве склада можно выбрать только тот склад, на котором используется ордерная схема отражения излишков и недостач товаров ( в карточке склада установлен флаг Использовать ордерную схему при отражении излишков и недостач ) .

Регистрация информации о порче товаров без оформления инвентаризации товаров (пересчета товаров)

Информация о таких товарах фиксируется в документе Ордер на отражение порчи товаров. В табличную часть добавляется список товаров, по которым обнаружена порча товаров (указывается товар исходного качества и испорченный товар). Информация о количестве бракованного товара вводится в графу Количество.

Данная операция может быть оформлена только на ордерном складе. В дальнейшем на основании этих данных оформляется финансовый документ Порча товаров. В случае неордерного склада порча товаров сразу фиксируется документом Порча товаров.

Приступить к регистрации документа можно с помощью команды Создать - Отражение порчи товаров.

Регистрация информации о недостаче товаров без оформления инвентаризации товаров (пересчета товаров)

Информация о таких товарах фиксируется в документе Ордер на отражение недостач товаров. В табличную часть добавляется список товаров, по которым обнаружена недостача (товар пропал во время перевозки, разбился и т.д.). Информация о количестве товаров, по которым обнаружена недостача вводится в графу Количество.

Данная операция может быть оформлена только на ордерном складе. В дальнейшем на основании этих данных оформляется финансовый документ Списание недостач товаров. В случае неордерного склада недостача товаров сразу фиксируется документом Списание недостач товаров.

Приступить к регистрации документа можно с помощью команды Создать - Отражение недостач товаров.

Регистрация информации об излишках товаров без оформления инвентаризации товаров (пересчета товаров)

Информация о таких товарах фиксируется в документе Ордер на отражение излишков товаров. В табличную часть добавляется список товаров, по которым обнаружены излишки. Информация о количестве товаров, по которым обнаружены излишки вводится в графу Количество.

Данная операция может быть оформлена только на ордерном складе. В дальнейшем на основании этих данных оформляется финансовый документ Оприходование излишков товаров. В случае неордерного склада излишки товаров сразу фиксируются документом Оприходование излишков товаров.

Данный документ не предназначен для оприходования начальных остатков товаров. Для этих целей предназначен документ Ввод начальных остатков товаров.

Приступить к регистрации документа можно с помощью команды Создать - Отражение излишков товаров.
