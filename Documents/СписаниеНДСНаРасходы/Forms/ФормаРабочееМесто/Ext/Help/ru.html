Рабочее место

Рабочее место предназначено для работы с документами списания НДС на расходы.

закладка Списание НДС

закладка Входящий НДС

закладка Списание НДС


На закладке отражается список документов с отбором по периоду и организации, указанным в рабочем месте.

закладка Входящий НДС


На закладке выводятся остатки НДС по документам-основаниям, по которым отсутствует счет-фактура некоторый период времени.
Период за который анализируется отсутствие счета-фактуры выбирается в соответствуеющем поле.

С помощью команды Списать НДС на основании документа-основания создается документ Списание НДС на расходы.
