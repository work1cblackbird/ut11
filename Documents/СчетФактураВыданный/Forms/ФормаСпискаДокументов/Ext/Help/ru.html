Форма списка документов

Форма предназначена для централизованной работы с выставлением счетов-фактур по операциям реализации.
Рабочее место позволяет проанализировать документы продажи, по которым не введены счета-фактуры, с возможностью непосредственного их оформления.

Оформление счет-фактуры
  • Перейдите на закладку К оформлению
  • Выделите строку таблицы с документом продажи, на основании которого необходимо создать счет-фактуру.
  • Нажмите на кнопку Оформить счет-фактуру
  • В появившейся диалоговой форме можно указать номера и даты платежно-расчетных документов, если была произведена предоплата.
  • Нажмите на кнопку Провести и закрыть для регистрации счета-фактуры.

Если при оформлении счета-фактуры выделить несколько строк документов продажи, то будет создан один счет-фактура на основании всех выбранных документы. Следует учитывать, что у документов продажи должны совпадать ключевые поля (Организация, Клиент, Валюта), иначе будет выдано предупреждение о невозможности создания счета-фактуры.

Группировка документов продажи


Документы продажи можно сгруппировать по одинаковым полям. Счет-фактура, оформленный по сгруппированной строке, будет содержать все входящие в неё документы-основания. Датой оформления такого счета-фактуры будет максимальная дата из входящих в неё документов.

Для настройки полей группировки воспользуйтесь гиперссылкой Настройка группировки документов-оснований в правом верхнем углу.

В сгруппированном виде список документов к оформлению будет иметь колонку Основания, в которой будет отражено количество документов-оснований по этой строке.

Сроки выставления счета-фактуры


Согласно п. 3 ст. 168 НК РФ счета-фактуры выставляются не позднее 5 календарных дней соответственно с момента отгрузки товаров (выполнения работ, оказания услуг), получения предоплаты, изменения стоимости товаров, работ, услуг. Документы, по которым со дня продажи прошло больше времени, отмечены предупреждающим цветом.
