Форма рабочее место для оформления счетов фактур комитенту

Рабочее место предусмотрено для перевыставления комитентам (принципалам) счетов-фактур, полученных комиссионерами (агентами) от поставщиков.
Перевыставление счетов-фактур оформляется документом Счет-фактура выданный. Счет-фактура выданный формируется по данным счетов-фактур, полученных от поставщиков.

Рабочее место доступно:

  • в панели навигации раздела Продажи по гиперссылке Оптовые продажи – Перевыставление комитенту (принципалу) счетов-фактур.
  • по гиперссылке Требуют переоформления/не оформлены, расположенной в подвале документа отчета комитенту (принципалу) о закупках.

На закладе К перевыставлению отображается список счетов-фактур, полученных от поставщиков, которые необходимо перевыставить комитенту (принципалу) по команде Перевыставить счет-фактуру. Для формирования/обновления счета-фактуры комитенту необходимо нажать команду Перевыставить счет-фактуру.

По результату перевыставления счета-фактуры в табличной части отчета комитенту (принципалу) заполняется автоматически счет-фактура выданный комитенту (принципалу) для каждой строки товара (колонка Счет-фактура выданный).

На закладке Перевыставленные счета-фактуры показывается список счетов-фактур, перевыставленных комитенту (принципалу).
