Форма списка документов

Для управления и контроля состояния документов по объединению записей складского журнала ВЕТИС доступны различные статусы, которые присваиваются документу автоматически.

Используя возможности фильтрации документов по объединению записей складского журнала в списке можно оставить только те документы, которые требуют определенных действий со стороны пользователя.

Для обмена данных с системой ВЕТИС предусмотрена команда Выполнить обмен. Список всех выполненных и предстоящих операций можно открыть по команде Протокол обмена.
