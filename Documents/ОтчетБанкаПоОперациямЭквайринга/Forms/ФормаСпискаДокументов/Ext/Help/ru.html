Основные поля документа


В списке отчетов банков по эквайрингу отображаются следующие данные:

  • Номер, Дата - номер и дата отчета по эквайринговым операциям, полученным из банка.
  • Договор экайринга - договор с банком, по которому ведутся расчеты.
  • Эквайрер - название банка, от которого получен отчет по эквайринговым операциям.
  • Сумма документа - общая сумма операций по платежным картам.
  • Проведено банком- признак проведения банком операции по зачислению денежных средств по платежным картам на расчетный счет организации. Устанавливается при включении отчета по эквайрингу в выписку банка.

Отбор отчетов по банку
  • Установите курсор в графу Эквайрер.
  • Нажмите на кнопку Найти.
  • Будет открыто диалоговое окно поиска.
  • В поле Где искать будет установлено Эквайрер.
  • В поле Что искать выберите банк, по которому Вы хотите посмотреть отчеты.
  • Будут показаны отчеты конкретного банка.


Можно провести быстрый поиск по названию банка. Для этого установите курсор в поле Эквайрер и начните набирать название банка. Диалоговое окно поиска при этом будет открыто автоматически.

Аналогичным образом можно произвести отбор по любому из полей, указанных в списке. Например, можно найти отчет по эквайринговым операциям по его номеру.
