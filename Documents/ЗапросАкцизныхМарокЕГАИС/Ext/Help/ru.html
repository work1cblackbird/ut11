Запрос акцизных марок ЕГАИС

Документ предназначен для запроса в ЕГАИС новых штрихкодов для поврежденных или нечитаемых акцизных марок, которые после проверки запроса должны быть нанесены на продукцию повторно. Запрос новых акцизных марок возможен только для организаций, деятельность которых связана с розничной продажей маркированной алкогольной продукции (имеется соответствующая лицензия).

В табличной части Товары указывается номенклатура и серия поврежденной (нечитаемой) акцизной марки. Данные для заполнения табличной части могут быть получены при чтении сканером штрихкода кода DataMatrix.

После получения из ЕГАИС новых штрихкодов их можно распечатать по кнопке Печать акцизных марок.

Статусы документа

  • Черновик - документ не выгружен в УТМ. В этом статусе можно производить любые операции над документом - изменение, передача в УТМ, удаление.
  • К передаче - документ поставлен в очередь на передачу в УТМ.
  • Передан в УТМ - документ передан в УТМ, получена квитанция из УТМ об успешной приемке.
  • Обрабатывается ЕГАИС - получена квитанция об успешной передаче документа в систему ЕГАИС.
  • Получены акцизные марки - из ЕГАИС загружены новые акцизные марки, доступна печать кодов марок в формате PDF417.
  • Ошибка получения акцизных марок - получена квитанция о неудачной попытке получения акцизных марок из ЕГАИС. В этом статусе можно производить любые операции над документом - изменение, повторная передача в УТМ, удаление.
