Корректировка регистров

Документ предназначен для ручной корректировки записей регистров накопления и регистров сведений.
По кнопке Настройка состава регистров открывается форма настройки состава регистров. Флажками отмечаются те регистры, записи по которым планируется ввести. На форме документа появляются закладки, соответствующие выбранным регистрам. В табличные поля на закладках вводятся необходимые данные в соответствии со структурой данных, предусмотренных в регистрах.

При проведении документа корректируется информация в тех регистрах, которые указаны в документе. С помощью гиперссылки Движения документа в панели навигации можно проконтролировать те движения, которые произведены после проведения документа.

Документ является служебным документом и доступен для заполнения только пользователю, который обладает полными правами (администратору системы).

Служебный реквизит Операция заполняется автоматически. Все корректировки, созданные пользователями вручную, имеют операцию Ручная корректировка. Остальные корректировки созданы системой.
