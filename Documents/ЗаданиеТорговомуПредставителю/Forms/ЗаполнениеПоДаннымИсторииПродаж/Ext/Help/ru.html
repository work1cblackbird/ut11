Анализ истории продаж клиента

Заполните информацию о периоде - нажмите на кнопку выбора интервала. В качестве интервала, за который будет выполняться анализ истории продаж товаров клиенту, можно указать предыдущий месяц, квартал, год и т.д. После выбора интервала будут заполнены конкретные даты. Даты начала и окончания периода анализа продаж могут быть выбраны и вручную. Если даты начала и окончания периода не указаны, то для анализа истории продаж используются данные о всех продажах клиента.

Способы расчета объема продаж (планируемого количества товаров)

При расчете объема продаж на основании истории продаж могут быть указаны различные способы расчета среднего объема за анализируемый период:

  • в среднем за день - объем вычисляется как средний за день. Данный способ может быть удобен, если задание формируется обобщенно, на день для клиента, который оформляет несколько продаж за день.
  • в среднем за неделю - объем вычисляется как средний за неделю. Данный способ может быть удобен при планировании заданий на неделю.
  • в среднем за месяц - объем вычисляется как средний за неделю. Данный способ может быть удобен при планировании заданий на месяц.

Флаг Детализировать по характеристикам

Устанавливать данный флаг следует в том случае, если необходимо, чтобы товарный состав задания, сформированный на основании истории продаж, был заполнен с учетом конкретных характеристик номенклатуры. В этом случае для товара в табличной части задания будет указана его характеристика и плановое количество товаров будет рассчитано отдельно для каждой его характеристики.

Если флаг не установлен, то информация о товаре заполнится без характеристик. При расчете планового количества будет рассчитано суммарное количество по всем товарам с характеристиками.

История продаж по ценовым группам

При заполнении данных на основе анализа истории продаж может быть задан отбор по номенклатуре, типам и видам номенклатуры. В этом случае в товарный состав задания будут включены только те данные истории продаж, которые удовлетворяют заданному отбору.

При заполнении истории продаж необходимо заполнять информацию только по тем товарам, которые принадлежат ценовой группе Холодильники. В этом случае устанавливаем флаг в поле Ценовые группы и выбираем ценовую группу Холодильники.

Аналогичным образом можно произвести отбор по определенной группе номенклатуры, виду или типу номенклатуры.
