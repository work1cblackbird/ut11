Корректировки обособленного учета запасов

Документ предназначен для отражения корректировок данных при ведении обособленного учета по сделкам, менеджерам или подразделениям.

Документ используется для обратной совместимости и в актуальных версиях не используется.

Для того чтобы зарегистрировать новую корректировку обособленного учета необходимо в списке корректировок воспользоваться командой Создать.
