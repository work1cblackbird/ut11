Чек ККМ

Документ предназначен для оформления продаж товаров в розницу с использованием Рабочего места кассира. Продажа товаров может производиться только с помощью кассы ККМ, в которой установлен тип кассы Фискальный регистратор и указан склад типа Розничный магазин. Расчет с покупателем производится наличными денежными средствами или платежной картой, подарочным сертификатом или бонусными баллами.
Для отдельных товаров (работ, услуг) в рамках соцобеспечения возможна оплата с использованием электронного сертификата (ЭС ФСС).

Подробное описание порядка оформления продаж в рабочем месте кассира приведено в документации на ИТС https://its.1c.ru/db/utdoc#bookmark:Retail:POS

Доступ к рабочему месту кассира может осуществляться с использованием персональных карт сотрудников со штрихкодом/магнитным кодом (при условии, что для данного рабочего места кассира в справочнике Настройка РМК в разделе НСИ и администрирование - РМК и оборудование - Рабочие места кассиров включена возможность Использовать авторизацию).
В этом случае работа с рабочим местом кассира будет возможна только при наличии у сотрудника персональной карты.

Оформление ручных скидок


После подбора товаров в документ кассир может указать для товара ручную скидку. Для этого предназначена кнопка Ручные скидки.

Ручная скидка в чеке может быть введена по всем позициям в чеке или по по конкретному товару. Ручная скидка может быть введена как процентная, так и суммовая.

Доступность ручных скидок определяетсмя функциональной опцией Ручные скидки в продажах в разделе НСИ и администрирование - CRM и маркетинг - Маркетинг.

Ручная скидка по всем позициям документа вводится в отдельном диалоговом окне, которое открывается при нажатии на кнопку Назначить ручную скидку. Для отмены ручных скидок используется команда Отменить ручные скидки.

Оформление автоматических скидок


Скидки, предусмотренные для розничного склада рассчитываются автоматически. Для просмотра информации о том, согласно каким условиям была зарегистрирована та или иная скидка, нажмите на гиперссылку с суммой рассчитанной скидки в информационной панели или выберите соответствующую команду в контекстном меню табличной части Товары. Будет показан список тех скидок которые применились для конкретного товара.

Расчет автоматических скидок необходимо проводить обязательно. Если скидки не рассчитаны, то чек не будет проведен.

Внесение денег


Опреация внесения денег в кассу ККМ осуществляется при помощи соответствующей команды из группы Операции с ККМ. При этом предварительно требуется оформить документ Расходный кассовый ордер с установленной операцией Выдача в кассу ККМ.

Пробитие чека


Пробитие чека следует производить после оформления всех операций по оплате чека. Для пробития чека на фискальном регистраторе нажмите на кнопку Пробить чек.

Печать товарного чека


Для печати товарного чека выполните следующие действия:

  • Выберите команду Товарный чек в меню, которое вызывается по кнопке Прочие операции.
  • Появится форма печати товарного чека.
  • Укажите количество копий печати товарного чека.
  • Нажмите на кнопку Печать.

Продажа алкогольной продукции


Если включена функциональная опция Алкогольная (спиртосодержащая) продукция в рознице и наступил срок оформления розничных продаж алкогольной продукции через ЕГАИС, то продажа маркированной алкогольной продукции осуществляется с обязательным указанием акцизных марок.

Перед выполнением продаж через ЕГАИС требуется настроить параметры подключения к ЕГАИС.

Оформление розничных продаж маркированной алкогольной продукции производиться только с помощью сканера штрихкодов, позволяющих считывать штриховые коды формата PDF417, размещенных на штрихкоде марке. При завершении оформления чека информация о продажах автоматически передается в ЕГАИС. После успешной печати фискального чека распечатывается слип-чек продажи алкогольной продукции с кодом подтверждения системы ЕГАИС. QR-код может быть распечатан в случае, если печать кода такого формата поддерживается фискальным регистратором. В противном случае код подтверждения будет выведен на печать в виде строки символов, что разрешено правилами оформления продаж ФСРАР. При потере связи с ЕГАИС оформление розничных продаж разрешается только в течение 3 дней.

Поведение системы при оформлении розничных продаж алкоголя изменяется в зависимости от настройки Дата начала регистрации розничных продаж в ЕГАИС. С момента наступления установленной даты для маркируемой алкогольной номенклатуры будет запрашиваться ввод кода марки в формате PDF417. Система также будет проверять, чтобы код марки был введен в чек однократно.

При считывании штрихкода алкогольной продукции запрашиваться считывание кода акцизной марки.
При считывании кода алкогольной продукции, если позицию номенклатуры по коду акцизной марки определить не удалось, то запрашивается считывание штрихкода номенклатуры.

Если выключена опция Продажа без сопоставления классификаторов с ЕГАИС, то реализация любой (маркированной и немаркированной) продукции возможна только после сопоставления классификаторов ЕГАИС и номенклатуры предприятия.

Удаление и редактирование акцизных марок производится по команде Акцизные марки.

Продажа табачной продукции


Если включена функциональная опция Вести учет табачной продукции и наступил срок оформления розничных продаж табачной продукции с обязательным сканированием марок, то продажа маркированной табачной продукции осуществляется с обязательным указанием кодов маркировки.

Перед выполнением продаж табачной продукции если включен обязательный контроль статусов кодов маркировки при розничных продажах, то требуется настроить сертификаты ЭЦП для авторизации в ИС МОТП.

Оформление розничных продаж маркированной табачной продукции производиться только с помощью сканера штрихкодов, позволяющих считывать штриховые коды формата DataMatrix. При завершении оформления чека информация о продажах автоматически передается в ОФД. ОФД передаст в ИС МОТП информацию о розничной продаже самостоятельно.

Поведение системы при оформлении розничных продаж табачной продукции изменяется в зависимости от настройки Дата ввода обязательной маркировки табачной продукции. С момента наступления установленной даты для маркируемой табачной продукции будет запрашиваться ввод кода маркировки. Система также будет проверять, чтобы код маркировки был введен в чек однократно.

Удаление и редактирование кодов маркировки табачной продукции производится по команде Акцизные марки.
