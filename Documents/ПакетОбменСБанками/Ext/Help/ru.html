Пакет прямого обмена с банками

Документ формируется автоматически и предназначен для хранения данных о взаимодействии с банками.

Для исходящих документов данные по Сообщению обмена заполняются в момент формирования пакета, для входящих документов — в момент распаковки, выполняемой либо автоматически программой, либо по команде Распаковать пакет (статус пакета должен быть К распаковке).

В некоторых случаях документ может быть использован службой технической поддержки для разбора проблемных ситуаций. Для этого необходимо выгрузить содержимое пакета по команде Сохранить пакет на диск.
