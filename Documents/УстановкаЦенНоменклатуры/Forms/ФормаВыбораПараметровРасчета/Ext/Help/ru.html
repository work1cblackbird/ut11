Ввод и регистрация документа


Нажмите на кнопку Создать. Появится новый документ, в котором необходимо указать следующие параметры:

  • Дату, начиная с которой цены начнут действовать.
  • Виды цен, по которым надо зарегистрировать значения цен.
  • Перечень номенклатурных позиций.
  • Для каждой номенклатурной позиции надо заполнить значение цены.


Для расчета цен не по всем, а только по отмеченным позициям, необходимо выполнить:

  • Выделить в списке те позиции, по которым надо рассчитать цены.
  • Нажать на кнопку Изменить цены.
  • Выбрать пункт меню Рассчитать вычисляемые (по данным ИБ и формулам). Появится диалоговое окно выбора параметров расчета цен.
  • В появившемся диалоговом окне установить переключатель Только выделенные;
  • Нажать на кнопку Рассчитать.


Аналогичным образом можно рассчитать цены только по тем позициям, по которым цены еще не рассчитаны. В этом случае в диалоговом окне настройки расчета цен необходимо установить переключатель Только незаполненные.

Установка статуса документа
  • Если в документе установлен статус Согласован, то его состояние в списке отображается как Действует.
  • Если в документе установлен статус Не согласован, то его состояние отображается как Ожидается согласование.
