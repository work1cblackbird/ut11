Подготовка заказов

На первом шаге необходимо сопоставить торговые предложения с номенклатурой и характеристикой номенклатуры, указать количество, выбрать организацию, от имени которой будут оформляться заказы.
Для перехода на следующий шаг нужно выбрать команду Подготовить заказы.

Редактирование списка товаров к заказу
  • Необходимо сопоставить торговые предложения с номенклатурой и характеристиками в информационной базе.
  • При необходимости изменить количество.
  • Управляя флагом в колонке К заказу, добавляется или исключается строка из заказа.
  • Для добавления торговых предложений к заказу выполнить команду Добавить, при этом откроется окно Поиска товаров по строке.
  • Для удаления торговых предложений из списка к оформлению заказа, выполнить команду Удалить текущий элемент.
  • Указать способ доставки товаров, адрес доставки.

Отправка заказов

Это второй и заключительный шаг помощника оформления заказов. На этом этапе рекомендуется указать контактную информацию для обратной связи: Контактное лицо, E-mail и телефон.
Для получения уведомлений по электронной почте об изменении статусов заказов необходимо установить флаг Уведомлять по эл. почте о статусе заказов.

Для отправки заказов через сервис 1С:Бизнес-сеть выполнить команду Отправить заказы поставщикам.
