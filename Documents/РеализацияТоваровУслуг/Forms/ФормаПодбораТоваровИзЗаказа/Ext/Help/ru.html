Форма подбора товаров и услуг из заказа

В форме отображается список товаров и услуг из заказов и расходных складских ордеров, по которым не оформлены реализации. Переключатель Все отгруженные/Все заказанные позволяет автоматически заполнять количество товаров, которые необходимо перенести в документ либо заказанными, либо отгруженными товарами. Необходимо флажками отметить те строки, которые необходимо перенести в документ. По команде Перенести в документ информация по отмеченным строкам будет обновлена в документе.

Для маркированного товара при заполнении по ордерам также заполняется информация по фактически подобранным маркам. При этом в строке для такого товара выводится стандартная пиктограмма обозначения маркируемого товара. Если при подборе найдены расхождения между количеством товара и подобранными марками, то пиктограмма розовая. Если расхождения не найдены, то пиктограмма зеленая.
