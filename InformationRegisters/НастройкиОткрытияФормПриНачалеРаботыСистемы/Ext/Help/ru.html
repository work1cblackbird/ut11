Настройки открытия форм при начале работы программы

Регистр предназначен для того, чтобы администратор мог определить для пользователя то рабочее место , которое будет открыто ему по умолчанию при начале работы.

Пример. Кассиру ККМ должно быть определено рабочее место кассира (форма чека ККМ) для оформления розничных продаж, а менеджеру по продажам удобно работать, используя обработку Помощник продаж.

В списке заполнена информация о всех зарегистрированных пользователях программы и рабочих местах (специализированных формах), которые могут быть им назначены в соответствии с установленными для них правами. Для установки рабочего места по умолчанию необходимо выполнить следующие действия.

  • Установить курсор на нужную запись в списке.
  • Выполнить команду Изменить.
  • В открывшейся форме установить флажок Открывать по умолчанию.
  • Для некоторых рабочих мест необходимо установить дополнительные параметры. Например, склад и разрешение экрана для рабочего места работника склада.

Совет. Возможен альтернативный способ для открытия рабочего места пользователю по умолчанию. Администратор может изменить у пользователя свойства ярлыка запуска 1С, указав нужные параметры запуска в соответствии с теми данными, которые указаны в списке Настройки открытия форм при начале работы с программой.
