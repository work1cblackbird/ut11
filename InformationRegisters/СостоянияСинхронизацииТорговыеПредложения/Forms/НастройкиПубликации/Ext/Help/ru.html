Уведомлять о заказах по эл.почте

При установке флага, на указанный адрес электронной почты будут приходить уведомленияо об отправке заказов товаров от покупателей.

Настройка режимов публикации
  • Публиковать цены на товары - выгрузка в сервис установленных цен на товары с возможностью просмотра покупателями. При снятии флага, в поле Цена будет указан текст По запросу. При этом, покупатель может выполнить заказ данного товара с последующей обработкой полученных заказов поставщиком, который должен установить цены для каждого заказа. Ответным документов на такой заказ может быть электронный документ Счет на оплату или Заказ покупателя.
  • Публиковать сроки поставки - выгрузка информации о сроках поставки товаров. При отключении, сроки поставки в торговых предложениях будут отображаться "по запросу".
  • Публиковать остатки товаров - выгрузка количества остатков товара в базовой единице измерения в сервис. Для покупателя количество отображается в терминах Неизвестно, Мало, Достаточно, Много.

Дополнительные условия продаж, отгрузки

В текстовом поле указывается произвольная информация об условиях продажи, скидках, отгрузки, доставки товаров. Информация отображается в форме просмотра карточки торгового предложения.
