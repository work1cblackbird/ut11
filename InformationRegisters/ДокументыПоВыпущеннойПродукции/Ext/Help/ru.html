Документы по выпущенной продукции для распределения

Данный регистр является техническим и используется для механизма партионного учета. Наличие записей в данном регистре является показателем что для данных документов необходимо выполнить списание затрат из регистра "Партии затрат на выпуск", которые пошли на данные документа выпуска/сборки
