Ведомость по партиям товаров

Отчет формируется по данным партионного учета и выводит остатки и движения партий товаров. На основании отображаемых данных можно контролировать поступление и списание партий товаров механизмом партионного учета и распределение расходов на партии товаров.

Общие принципы работы с отчетами описаны в документации на ИТС https://its.1c.ru/db/utdoc#bookmark:Reports:ReportSetup.

Данные отчета доступны только при включенной функциональной опции Партионный учет в разделе НСИ и администрирование - Финансовый результат и контроллинг - Учет товаров.
При использовании партионного учета версии 2.2 документ поступления заполняется только при методе оценки стоимости ФИФО (скользящая оценка).

Варианты формирования

Ведомость по партиям товаров предприятия

Отчет выводит остатки и движения товаров по предприятию в валюте управленческого учета с НДС или без НДС (параметр Данные себестоимости). В отчет выводится "сквозная" себестоимость товаров по предприятию в целом.

Ведомость по партиям товаров организаций

Отчет выводит остатки и движения товаров по организациям в валюте регламентированного или управленческого учета без НДС (параметр Данные себестоимости).

Особенности отчета:

  • Расшифровка данных по статьям расхода/комплектующим, а также документам формирования расходов;
  • Опциональная возможность исключения из отчета данных внутренних перемещений товаров между складами предприятия (флаг Исключить перемещения товаров);
  • Опциональная возможность добавления в отчет показателей Стоимость поступления и Дополнительные расходы;
  • Оценка себестоимости как с учетом НДС, так и без учета НДС, а также на основании данных регламентированного учета.
  • При формировании отчета контролируется актуальность распределения партий. Если данные неактуальны, то отчет формируется, в него выводится предупреждение и запускается фоновое задание по актуализации. После завершения данного задания будет предложено переформировать отчет.

В настройках отчета в режиме Расширенный предусмотрен параметр Количественные итоги. По умолчанию он принимает значение С учетом единиц измерения (по умолчанию). В этом случае итоги рассчитываются только, если все значения единиц измерения одинаковые в пределах группировки. Это позволяет исключить суммирование несопоставимых единиц измерения, например количественных и метрических. При установки настройки Количественные итоги установить в значение Без учета единиц измерения групповые итоги сформируются путем сложения количества по всем позициям, без учета их единиц измерения.
