Товары схемы обеспечения

Отчет предназначен для анализа способов обеспечения товаров по складам.

Особенности отчета:

  • Индивидуальные настройки способа обеспечения оформляются синим цветом текста.
  • Характеристики номенклатуры выводятся в отчет только в том случае, если для них установлены индивидуальные настройки способа обеспечения.
  • Расшифровка по способу обеспечения открывает рабочее место Настройка способов обеспечения потребностей с установленным отбором по товару и позиционированием по складу.
