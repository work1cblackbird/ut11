Сравнение сегментов клиентов

Отчет предназначен для сравнения деятельности партнеров, которые принадлежат к различным сегментам. Для анализа деятельности партнеров используются как абсолютные, так и удельные показатели. Удельные показатели рассчитываются по отношению к тому количеству партнеров, которые входят в сегмент.

Пример. Количество партнеров в сегменте равно 100. Партнерами данного сегмента было заключено 20 сделок. Удельный показатель будет равен 0,2 ( 20/100 ).

Отчет содержит таблицу по сегментам с указанием названия сегмента и количества партнеров, входящих в сегмент. По каждому сегменту выводятся следующие абсолютные и удельные показатели:

  • Количество незавершенных сделок - количество сделок, зарегистрированных с партнерами данного сегмента, для которых установлен статус В работе.
  • Количество проигранных сделок - количество сделок, зарегистрированных с партнерами данного сегмента, для которых установлен статус Проиграна.
  • Количество выигранных сделок - количество сделок, зарегистрированных с партнерами данного сегмента, для которых установлен статус Выиграна.
  • Сохраненный потенциал - сумма потенциального объема продаж, который зарегистрирован в выигранных сделках.
  • Потерянный потенциал - сумма потенциального объема продаж, который зарегистрирован в проигранных сделках.
  • Потенциал незавершенных сделок - сумма потенциального объема продаж, который зарегистрирован в сделках, которые находятся в работе.
  • Объем продаж - суммарный объем продаж по всем партнерам, входящим в сегмент.
