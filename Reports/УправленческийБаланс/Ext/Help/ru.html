Управленческий баланс

Отчет предназначен для анализа финансового состояния предприятия с контролем баланса активов и пассивов.

Отчет доступен, если включена опция НСИ и администрирование - Финансовый результат и контроллинг - Формировать управленческий баланс.

Показателями отчета являются статьи активов и пассивов.
Поддерживается произвольная настройка иерархии групп и приоритета расположения статей в справочнике Статьи активов и пассивов.
Для учета прочих активов и пассивов возможно пополнение справочника статей требуемыми для детализации управленческого учета статьями.

Данные по статьям активов и пассивов хранятся только в управленческой валюте.
Выбор валюты отчета влияет на вывод всех сумм в требуемой валюте пересчетом по курсу на дату отчета. Это может давать отклонения из-за округлений по показателям.

Варианты формирования:

Управленческий баланс

Управленческий баланс по направлениям

Управленческий баланс по организациям

Варианты формирования:

Управленческий баланс

Управленческий баланс с группировкой по временным периодам.

Особенности отчета:

  • группировка по временным периодам;
  • без аналитического разреза по организациям и направлениям деятельности;
  • организации (подразделения) и направления деятельности могут служить отбором;
  • данные выводятся по фиксированному временному периоду (месяц, квартал, полугодие, год).


Для показа двух и более колонок с динамикой "было-стало" по периодам нужно выбрать даты отчета на несколько периодов, кратно выбранному типу периодичности.
Для показов изменений в периоде (то есть оборотов прихода и расхода по показателям) нужно выбрать настройки отчета и отметить режим Выводить обороты по статьям.
По умолчанию не на всех статьях включен показ оборотов, а на некоторых основных.
Если нужно на всех (или недостающих для анализа), то в карточке статей активов/пассивов нужно поставить отметку Выводить обороты в управленческом балансе.

Управленческий баланс по направлениям

Управленческий баланс с группировкой по направлениям деятельности.

Детализация по направлениям деятельности в отчете возможна только при включенной опции НСИ и администрирование - Финансовый результат и контроллинг - Финансовый результат по направлениям деятельности и настройках опций обособленного учета доходов и расходов в справочнике Направления деятельности.

Особенности отчета:

  • группировка по направлениям деятельности;
  • без аналитического разреза по организациям;
  • организации и подразделения могут служить отбором, но существует риск нарушения контроля баланса.

Управленческий баланс по организациям

Управленческий баланс с группировкой по организациям.

Особенности отчета:

  • группировка по организациям;
  • без аналитического разреза по направлениям;
  • направления деятельности и организации могут служить отбором, но существует риск нарушения контроля баланса.


Использование документов интеркампани приводит к разрывам баланса при использовании отборов по организациям. Эти документы в балансе учитываются сквозным образом по всем организациям одновременно.

Для расшифровки показателей управленческого баланса используются отчеты:

  • Ведомость расчетов с клиентами
  • Ведомость расчетов с поставщиками
  • Ведомость по денежным средствам
  • Денежные средства у подотчетных лиц
  • Доходы и расходы
  • Ведомость расчетов по финансовым инструментам
  • Расчеты между организациями
  • Себестоимость товаров
  • Состояние расчетов с комитентами
  • Оборотно-сальдовая ведомость по статьям активов и пассивов


Отчет входит в группу отчетов Финансовый результат:

  • Валовая прибыль предприятия,
  • Валовая прибыль по полностью оплаченным отгрузкам,
  • Валовая прибыль по частично оплаченным отгрузкам,
  • Доходы и расходы,
  • Финансовые результаты.
