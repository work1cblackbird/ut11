Управленческий баланс (контроль)

Отчет предназначен для контроля нарушений баланса в различных периодах отдельно взятыми документами.

Принцип контроля основан на том, что на момент построения отчета (после закрытия месяца) внутри документа-регистратора уже хранятся все управленческие движения по статьям активов и пассивов, обеспечивающие баланс по двойной записи (сумма всех приходов равна сумме всех расходов).

Проводки управленческого учета при этом формируются по статьям активов/пассивов по принципу "сложных проводок", то есть на один Расход (Кт) может быть несколько Приходов (Дт), либо наоборот.

При анализе разрывов баланса вызовом отчета, как расшифровки из управленческого баланса по итоговой строке, нужно учитывать, что отчет показывает данные за период. Сам же управленческий баланс показывает остатки на отчетную дату. Разрыв возможен в предыдущих периодах. Поэтому контроль баланса нужно строить с большей временной глубиной, если искомые разрывы в текущем периоде не показываются.
