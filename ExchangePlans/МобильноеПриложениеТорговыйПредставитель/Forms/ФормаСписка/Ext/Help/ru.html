Настройки торговых представителей

Определяет настройки пользователей мобильного приложения (торговых представителей): отбор данных, участвующих в обмене, а также некоторые параметры управления обменом.

Сколько настроек обмена должно быть создано в списке, если торговый представитель использует несколько мобильных устройств?

В список вводятся настройки обмена для каждого мобильного устройства, с которым работает торговый представитель. То есть, если один торговый представитель использует два мобильных компьютера, то необходимо ввести в список информацию о двух настройках обмена с каждым мобильным компьютером.

При обмене данными идентификация настроек производится по имени пользователя (торгового представителя), указанному паролю и серийному номеру мобильного компьютера. При использовании в качестве мобильного устройства персонального компьютера для идентификации используется сетевое имя компьютера.

Как добавить новую настройку для обмена с мобильным устройством?

В списке настроек обмена мобильных пользователей нажмите кнопку "Создать" в панели управления списком. Откроется карточка новой настройки, в которую необходимо ввести все необходимые данные.
